Rationale

Implemented

Colour:

The soft colours chosen help the site to be readable during the day and night without straining the eyes.

Fixed Header/Navigation Bar:

In the default stylesheet the Header and Navigation bar are fixed, so this would help the user to have the ability to move through the site quickly. Also this helps to utilize more of the width of the website to show more content horizontally.

Projection Style:

This stylesheet helps to display more content and with bigger font size for projectors. The Header and Navigation bar are reduced in size and not fixed, since it will allow to show more content.

Printer Style:

It removes all the colours and unneeded elements like header, navigation bars and reduces blank spaces in order to waste less ink.

Horizontal Menu:

Having all the pages in an horizontal design, allows more space for the content to be shown horizontally. Also, if the window is compressed the menu will adapt in several rows.


Pending

Mobile Style:

For the students that would like to read before the class or study for the quizzes, it would be helpful to have the contect adapted to small or medium screens in landscape or portrait mode.

Submenus:

In order to allow more links in the menu would be perfect to include the submenus functionality, this would show another row with more links, and hide the siblings on the previous link selected.
